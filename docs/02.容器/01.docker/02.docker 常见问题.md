> 1. apline 镜像时区
**（1）带参数运行**
```
docker run -d -e "TZ=Asia/Hong_Kong"
```
**（2）修改镜像**
```
FROM imageName:apline
RUN ln -sf /usr/share/zoneinfo/Asia/Hong_Kong /etc/localtime
```
```
docker build -t imageName:hongkong-timezone .
```
